# vgmusic-downloader

Download all MIDIs from VGMusic!

## Dependecies

```
pip install -r requirements.txt
```

## Usage

```
python vgmusic.py <vgmusic_console_page>
```

### Example:

```
python vgmusic.py https://www.vgmusic.com/music/console/nintendo/snes/
```
